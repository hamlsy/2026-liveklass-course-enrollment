# LiveKlass Backend Assignment: Course Enrollment System

## 프로젝트 개요
* 서비스명: LiveKlass 수강 신청 마이크로서비스
* 목적: 크리에이터의 강의 개설 및 클래스메이트의 선착순 수강 신청, 결제, 취소 로직 구현
* 핵심 목표: 대량의 동시성 요청 상황에서 데이터 정합성을 유지하며 안정적인 수강 신청 처리

## 기술 스택
* Framework: Spring Boot 4.0.5
* Language: Java 17
* Database: H2
* Persistence: Spring Data JPA
* Concurrency: DB (Pessimistic Lock)
* Build Tool: Gradle
